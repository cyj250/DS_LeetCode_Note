**题目描述**

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

示例:

> 输入: [0,1,0,3,12]
> 输出: [1,3,12,0,0
>
> 

说明:

1. 必须在原数组上操作，不能拷贝额外的数组。
2. 尽量减少操作次数。

**方法一**

首先遍历整个数组，将非零元素移动的前边，再将后边置为零

代码如下：

```c
void moveZeroes(int* nums, int numsSize){
    int index=0,i;
    for(i=0;i<numsSize;i++){
        if(nums[i]){
            nums[index++] = nums[i];
        }
    }
    for(i=index;i<numsSize;i++){
        nums[i] = 0;
    }
}
```

**方法二**

设两个指针*left*和*right*，right向右遍历，寻找非零的元素，left固定在为零的元素处，然后交换left和right的元素，left++，right++；

![283_1.gif](https://pic.leetcode-cn.com/9669b4ffb158eaeeee6f0cd66a70f24411575edab1ab8a037c4c9084b1c743f5-283_1.gif)

```c
void moveZeroes(int* nums, int numsSize){
   int left=0,right = 0;
    while(right<numsSize){
        if(nums[right]){
            int tem = nums[left];
            nums[left++] = nums[right];
            nums[right] = tem;
        }
        right++;
    }
}
```
